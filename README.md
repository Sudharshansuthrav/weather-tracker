# Weather Tracker 🌦️

## 📌 Overview

Weather Tracker is a web-based application that allows users to check real-time weather updates for any location. It fetches data from a weather API and displays information such as temperature, humidity, wind speed, and weather conditions.

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Display real-time temperature, humidity, and wind speed
- 🌤️ Dynamic weather icons based on conditions
- 📊 User-friendly and responsive UI

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **API:** OpenWeatherMap API
- **Tools:** Git, VS Code

## 🔧 Usage

1. Enter the name of a city in the search bar.
2. View real-time weather data including temperature, humidity, and wind speed.
3. Get weather condition icons that dynamically update.
